Place UTF-8 capable TTF fonts in this folder to enable full Unicode PDF output.

Supported filenames (first match wins):

- NotoSansSC-Regular.ttf / NotoSansSC-Bold.ttf
- SourceHanSansSC-Regular.ttf / SourceHanSansSC-Bold.ttf
- LXGWWenKai-Regular.ttf / LXGWWenKai-Bold.ttf

You can also provide custom paths via environment variables:

- AGENTFLOW_INVOICE_FONT_REGULAR
- AGENTFLOW_INVOICE_FONT_BOLD

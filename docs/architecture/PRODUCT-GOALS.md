# 产品目标与范围定义

版本：v1.0
日期：2026-02-03
状态：Active

---

## 1. 产品目标

### 核心愿景

用户可以使用 AI Work Flow 构建应用并托管在平台上，实现从需求描述到可运行应用的完整闭环。

### 目标分解

| 目标类别 | 目标描述                                         | 验收标准                             |
| -------- | ------------------------------------------------ | ------------------------------------ |
| 构建能力 | 用户可以通过 AI 对话或可视化编辑器创建工作流应用 | 支持 AI 生成 + 手动编辑双入口        |
| 运行能力 | 应用可以被执行、调度、监控                       | 支持即时执行、定时触发、执行历史查看 |
| 发布能力 | 应用可以发布为公开访问的服务                     | 支持公开链接、匿名访问、自定义域名   |

### 产品定位

- **目标用户**：有自动化需求的个人用户、小团队、企业开发者
- **核心场景**：
  - 快速构建 AI 驱动的自动化工作流
  - 将工作流打包为可对外访问的应用
  - 团队协作管理多个应用和工作流

## 2. 核心能力清单

| 能力           | 描述                                  | 优先级 |
| -------------- | ------------------------------------- | ------ |
| 默认 Workspace | 每个用户自动拥有一个默认工作空间      | P0     |
| 多应用管理     | 一个 Workspace 可创建和管理多个应用   | P0     |
| 公开访问       | 应用可发布为公开可访问的服务          | P0     |
| 可选匿名       | 支持按 App 级别开启/关闭匿名访问      | P0     |
| 独立数据库     | 每个 Workspace 可拥有独立的数据库实例 | P1     |
| 域名绑定       | 应用支持绑定自定义域名                | P1     |

### 能力与需求映射

```
需求：用户可以创建和管理工作空间
  └─ 能力：默认 Workspace ✓
  └─ 能力：Workspace 设置 ✓
  └─ 能力：成员管理 ✓

需求：用户可以创建和发布应用
  └─ 能力：多应用管理 ✓
  └─ 能力：AI 生成应用 ✓
  └─ 能力：应用版本管理 ✓

需求：应用可以被公开访问
  └─ 能力：公开访问 ✓
  └─ 能力：可选匿名 ✓
  └─ 能力：域名绑定 ✓

需求：数据安全隔离
  └─ 能力：独立数据库 ✓
  └─ 能力：权限控制 ✓
```

## 3. 边界与非目标

### 明确的非目标（当前版本不做）

| 非目标                | 原因                 | 未来考虑 |
| --------------------- | -------------------- | -------- |
| 全托管云函数          | 复杂度高、安全风险大 | Phase 3+ |
| 外部自定义运行时      | 需要沙箱隔离方案     | Phase 4+ |
| 实时协作编辑          | 工程复杂度高         | Phase 2+ |
| 跨 Workspace 数据共享 | 安全与隔离冲突       | 暂不考虑 |
| 私有部署版本          | 资源限制             | 商业化后 |

### 范围边界

```
┌──────────────────────────────────────────────────────────┐
│                    平台范围内（In Scope）                  │
├──────────────────────────────────────────────────────────┤
│  ✓ Workspace 管理与团队协作                               │
│  ✓ App 生命周期管理（创建、编辑、发布、归档）               │
│  ✓ Workflow 可视化编辑与执行                              │
│  ✓ AI 辅助生成工作流和 UI                                 │
│  ✓ 公开访问与匿名会话                                     │
│  ✓ 自定义域名与 TLS 证书                                  │
│  ✓ Workspace 独立数据库                                   │
│  ✓ 用量计量与配额控制                                     │
│  ✓ 基础监控与日志                                        │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│                   平台范围外（Out of Scope）               │
├──────────────────────────────────────────────────────────┤
│  ✗ 用户自定义运行时/容器                                  │
│  ✗ 全功能云函数平台                                      │
│  ✗ 跨租户数据访问                                        │
│  ✗ 实时多人协作编辑                                      │
│  ✗ 私有化部署                                           │
└──────────────────────────────────────────────────────────┘
```
